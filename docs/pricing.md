## Hardware COGS — China FOB landed cost model

Every physical component is China-sourced. Customer prices are UK prices (ex-VAT for B2B).

**Landed COGS formula:**
> China FOB + freight + import duty + customs/handling + QC/testing + packaging + warranty reserve + UK fulfilment

| Cost item | Planning rate |
|---|---|
| Freight / logistics | 8–15% of FOB for bulk; higher for samples |
| Import duty (electronics HS code) | 0–3% |
| Customs / admin / inbound handling | 3–5% |
| QC / testing | 5–8% |
| Warranty reserve | 5–10% |
| UK fulfilment / packing | £4–£10 per order |

> A $25–$35 China factory device realistically becomes **£40–£50 landed COGS** in the UK.

## Break-even

- Tier 1 contribution margin: £63 − £13 = **£50/site/month**
- Break-even: £20,000 ÷ £50 = **~400 paying sites**
- That is **< 9% of London's ~4,500 independent cafés alone**
