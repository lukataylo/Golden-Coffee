## Hardware COGS — China FOB model

All physical units priced as **FOB + freight + import + QC + warranty + UK fulfilment**.
A $25–$35 China factory device lands at ~£42–£50 UK COGS after uplift:

| Cost item | Planning rate |
|---|---|
| Freight / logistics | 8–15% of FOB |
| Import duty (electronics) | 0–3% |
| Customs / admin | 3–5% |
| QC / testing | 5–8% |
| Warranty reserve | 5–10% |
| UK fulfilment | £4–£10/order |

Steve sensor (ESP32-S3 + environment module + enclosure): **FOB ~£26–£32 → landed ~£42–£48**.

## Break-even

Fixed cost base (lean London startup, 3 engineers + ops + office): **~£18–£25k/month**.

| Active paying locations | Fixed cost per customer |
|---:|---:|
| 100 | £200/mo — **not viable standalone** |
| 300 | £67/mo — approaching break-even |
| 500 | £40/mo — **profitable** |
| 1,000 | £20/mo — strong margin |

> Tier 1 at 500 sites = **£31,500 MRR** against ~£6,500 variable COGS → ~£25k contribution.
> Tier 2/3 mix accelerates this significantly.
